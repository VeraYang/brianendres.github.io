HTML

div

-The div element is used as a divider to separate peices of content from eachother.

-It is important becuase it allows you to choose which part of a page a certain element appears on. Also it gives you a container to hold elements

p

-The paragraph element in html is used to hold text, or a paragraph of text

-It is important becuase it allows you to add peices of text to your website

CSS

Block

-A block element starts a new line and takes up all the availible width.

-An inline element does not start a new line and only takes up enough space to fit the content, or a set width.

-The display property allows you to change inline elements to block

Box Model

-The box model system allows you to determine how your content is placed inside a container, and how things around it will be moved

-Padding allows you to set the distance between your content and the containter around it

-Margin allows you to set the distance between your container and the things outside of it

-Border allows you to set a border style, width, and color that goes along the edge of your container

Float/Clear

-Float allows you to move an element to the left or right side of the page

-Clear allows you to decide where the element will be placed around other elements
